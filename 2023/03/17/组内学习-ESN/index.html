ESN基本介绍

  • 循环神经网络

    • 概述

      RNN解决了标准神经网络中input，output大小不相同，以及不能共享学习到的不同位置的数据内容。而且标准的神经网络对数据预测的话，会需要非常多的参数。

    • RNN的基本流程：

      RNN在每一个时间步上输入x输出y，传递参数给下一个时间步，它的时间步参数是共享的。

    • RNN的前向传播与反向传播

    • 几种结构

      many-to-many

      many-to-one

      one-to-many

      注意力机制（…）

      需要一个语料库，用onehot编码标注所有句子里的单词，再加上一个结尾标记EOS。如果有语料库没有的单词，打上标记UNK

  • 回声状态网络

    • 概述

      • 属于RNN，处理序列化的数据，且能够捕捉长距离的依赖关系。

      • 回声状态网络又称储备池计算（RC，reservoir computing），采用由随机生成稀疏连接固定不变的内部权重矩阵（$W$）得到神经元组成的储备池作为隐藏层。ESN将神经网络的隐层权值预先生成而非训练生成，与隐藏层到输出层的权值训练分开进行，其基本思想的前提是生成的储备池具有某种良好的属性。

        ESN通过修改输出层的权重来调整模型，通常采用线性回归或其他监督学习技术来优化输出层的权重。一般来说，ESN的训练速度比传统的循环神经网络更快，而且在时间序列预测和分类等任务上具有很好的性能。

        与传统的RNN相比，ESN具有快速训练、高效率、易于实现等优点。但是，ESN也存在一些问题，例如由于其随机性，可能会出现过拟合等情况，需要在实际应用中进行有效的优化和改进。

    • 网络结构

      • 输入层，储备池（隐藏层），输出层

      • 输入单元，储备池单元和输出单元

      • 该网络使用输入信号序列[$u_{1}(t)$, … ,$u_{K}(t)$]以及target序列[$y_{1}(t)$, … ,$y_{L}(t)$]来训练模型，输出预测的$y_{L+1}(t)$，$y_{L+2}(t)$，$y_{L+3}(t)$等数据

    • ESN类方法—初始化权重initweights

      这个函数是在静态属性里面被调用的，也就是说创建类的时候已经初始化完成。

      一共有四个矩阵$W_{in}$ ,$W$,$W_{out}$ ，$W_{feedback}$初始化部分主要针对$W$,$W_{in}$ ,$$W_{feedback}$$这三个矩阵

      K为 n_inputs = 1

      N为 n_reservoir = n_reservoir

      M为 n_outputs = 1

      • 初始化$W$矩阵（N*N）

        • dropout操作中self.random_state_.rand(*W.shape) < self.sparsity是一个mask矩阵，里面的元素是bool类型。True的话相应元素就能赋值为相应值，用mask矩阵与原矩阵相乘也能达到这个目的。
        • np.linalg.eigvals(W)求特征值，最后取绝对值最大的特征值，即谱半径，因为谱半径是指其特征值绝对值集合的上确界.
        • 原矩阵除以谱半径得到最后初始化的矩阵$W$

      • 初始化$W_{in}$矩阵 (N*K)

      • 初始化$W_{feedback}$矩阵 (N*M)

    • ESN类方法—训练ESN网络fit

      训练部分包含了好几个ESN类方法，_ update，_ scale_inputs，_ scale_teacher，_ unscale_teacher

      • 输入量为np.ones(trainlen),data[i:trainlen+i] 作为训练的input和output,还有一个无关紧要的insepct参数使得储备池收集可视化。

        其中input为 (数据维度trainlen x 输入维度K)；output为 (数据维度trainlen x 输出维度M)

      • 以下分段说明步骤：

        • 传入数据标准化

          由于一维数组的形状是（x,）改正为(x,1)

        • 增加偏移值或者乘以参数（*）

        • 更新state数组（使用input来更新）

        • 扩展state数组，加入原始数据input,变成extended_states。

        • 求解$W_{out}$

        • 记录最后的state ， input 和 output

      • ```python
        def fit(self, inputs, outputs, inspect=False):
        “””
        Collect the network’s reaction to training data, train readout weights.

        Args:
        inputs: array of dimensions (N_training_samples x n_inputs)
        outputs: array of dimension (N_training_samples x n_outputs)
        inspect: show a visualisation of the collected reservoir states

        Returns:
        the network’s output on the training data, using the trained weights
        “””

        # 将传入的数据标准化
        # 比如，把一维向量(11,)更正为 (11, 1):
        if inputs.ndim < 2:
        inputs = np.reshape(inputs, (len(inputs), -1))
        if outputs.ndim < 2:
        outputs = np.reshape(outputs, (len(outputs), -1))

        #增加偏移值或乘以参数（_scale_inputs，_scale_teacher）
        # transform input and teacher signal:
        inputs_scaled = self._scale_inputs(inputs)
        teachers_scaled = self._scale_teacher(outputs)

        #注释神经网络运行状态(获取储备池状态)
        if not self.silent:
        print(“harvesting states…”)

        #更新state数组(类似初始化)
        # step the reservoir through the given input,output pairs:
        states = np.zeros((inputs.shape[0], self.n_reservoir))
        print(“inputs.shape”,inputs.shape)
        for n in range(1, inputs.shape[0]):
        states[n, :] = self._update(states[n - 1], inputs_scaled[n, :],
        teachers_scaled[n - 1, :])
        # learn the weights, i.e. find the linear combination of collected
        # network states that is closest to the target output
        # 注释神经网络运行状态(获取储备池状态)
        if not self.silent:
        print(“fitting…”)

        # we’ll disregard the first few states:
        # 不考虑前几个状态（阈值）
        transient = min(int(inputs.shape[1] / 10), 100)
        #扩展state，加入原始数据（state之前是在初始化的赋值）
        # include the raw inputs:
        extended_states = np.hstack((states, inputs_scaled))
        #求解Wout
        # Solve for W_out:
        self.W_out = np.dot(np.linalg.pinv(extended_states[transient:, :]),
        self.inverse_out_activation(teachers_scaled[transient:, :])).T

        #记录最后训练的数据state , input 和 output
        # remember the last state for later:
        self.laststate = states[-1, :]
        self.lastinput = inputs[-1, :]
        self.lastoutput = teachers_scaled[-1, :]

        # optionally visualize the collected states
        if inspect:
        from matplotlib import pyplot as plt
        # (^– we depend on matplotlib only if this option is used)
        plt.figure(
        figsize=(states.shape[0] * 0.0025, states.shape[1] * 0.01))
        plt.imshow(extended_states.T, aspect=’auto’,
        interpolation=’nearest’)
        plt.colorbar()

        # 注释神经网络运行状态(打印训练误差)
        if not self.silent:
        print(“training error:”)

        #将求解得到的数组返回到state里面，训练完成更新
        # apply learned weights to the collected states:
        pred_train = self._unscale_teacher(self.out_activation(
        np.dot(extended_states, self.W_out.T)))

        # 注释神经网络运行状态(打印MSE结果)
        if not self.silent:
        print(np.sqrt(np.mean((pred_train - outputs)**2)))

        return pred_train
